# Endless

**Финансовый автопилот**, который делает всё сам и дёргает тебя только когда нужно.

Endless — это AI-надстройка над [Дзен-мани](https://zenmoney.ru/), которая автоматически категоризирует транзакции, показывает куда уходят деньги, и помогает накапливать — без ручного ввода и ежедневной рутины.

## Принципы

<ExampleBox symbol="🤖">
  **Zero friction.** Система работает в фоне. Нет «обязательных» действий. Пропустил день — ничего не сломалось.
</ExampleBox>

<ExampleBox symbol="⚡">
  **30 секунд в день.** Утром приходит дайджест: вчерашние расходы, остаток по конвертам, один инсайт. Глянул — пошёл дальше.
</ExampleBox>

<ExampleBox symbol="🧠">
  **AI-first.** 5-уровневый категоризатор: твои правки → паттерны → комментарии → MCC-коды → LLM. 90%+ транзакций размечаются без тебя.
</ExampleBox>

<ExampleBox symbol="👆">
  **Human-in-the-loop.** AI спрашивает только про то, что не понимает. Каждая твоя правка обучает систему — в следующий раз она уже знает.
</ExampleBox>

## Метод

Endless объединяет четыре подхода, подтверждённых исследованиями:

### От Вавилона — «Заплати себе первому»
Минимум 30-40% дохода автоматически уходит в накопления и инвестиции **до** расходов. Не «что останется», а «сначала себе, потом всё остальное». Программа Save More Tomorrow Талера и Бенарци показала рост сбережений с 3.5% до 13.6% — просто за счёт автоматизации.

### От Кийосаки — активы работают
Единственная метрика, которая имеет значение: **passive income / total expenses**. Дашборд показывает процент движения к точке, где пассивный доход покрывает расходы.

### От YNAB — каждый рубль распределён
Zero-based budgeting: деньги приходят → раскладываются по конвертам → нераспределённых = 0. Если перетратил в одной категории — перекладываешь из другой, а не «всё сломано, начинаю заново».

### От поведенческой науки — система без воли
Дефолты вместо решений. Автоматизация вместо дисциплины. Мягкие подсказки вместо красных баннеров «БЮДЖЕТ ПРЕВЫШЕН». Гибкий бюджет вместо жёстких правил — чтобы срыв становился адаптацией, а не поводом бросить.

## Как это работает

### 1. Подключение
Endless работает поверх Дзен-мани через API. Все данные остаются в Дзен-мани — Endless только читает транзакции и пишет категории.

### 2. AI-категоризация
При первом запуске AI проходит по всей истории и размечает транзакции. Система знает российских мерчантов (Пятёрочка, Яндекс.Такси, Ozon...), читает MCC-коды, анализирует комментарии. Что не понимает — спрашивает.

### 3. Ежедневный автопилот
Каждый день система:
- Подтягивает новые транзакции
- Категоризирует автоматически (>85% уверенности — без вопросов)
- Спрашивает по переводам людям (пока помнишь кому и зачем)
- Обновляет дашборд

### 4. Конверты (бюджеты)
Зарплата пришла → система предлагает распределение:
- 30% → Инвестиции
- 10% → Подушка / цели
- 33% → Фиксированные расходы
- 27% → Свободные траты

Можешь подвинуть пропорции. Главное — инвестиции уходят **первыми**, а не «что останется».

### 5. Аналитика
- **Куда уходят деньги** — расходы по категориям с мини-трендами по месяцам
- **По месяцам** — доходы vs расходы, savings rate, динамика
- **Конверты** — сколько осталось до конца месяца в каждой категории

## Ключевые метрики

| Метрика | Зачем |
|---------|-------|
| **Savings rate** | Реальные переводы на сбер. счета ÷ доход |
| **Runway** | На сколько месяцев хватит при текущих расходах |
| **Ликвидные активы** | Наличные и вклады (без кредитов/ипотеки) |
| **Чистый капитал 10 лет** | Прогноз с 3 сценариями (🌧/📊/🚀) |
| **До зарплаты осталось** | Сколько денег до следующей ЗП, с учётом кредитов |
| **Подушка / FIRE** | Прогресс к цели: 6 мес подушка, 25× FIRE |

## Чего Endless НЕ делает

- Не показывает красные баннеры «ПЕРЕРАСХОД» — вместо этого мягко предлагает перебалансировать
- Не требует ежедневного ручного ввода — AI делает 90%+
- Не наказывает за срывы — бюджет адаптируется, а не ломается
- Не считает vanity-метрики (кол-во транзакций, дни в приложении)

## Безопасность

<ExampleBox symbol="🔒">
  Endless не имеет доступа к банковским подключениям. Данные хранятся локально и в Дзен-мани. Токен используется только для чтения транзакций и записи категорий через [ZenMoney API](https://github.com/zenmoney/ZenPlugins/wiki/ZenMoney-API).
</ExampleBox>
